# Repository Overview

This repository is a long-form Java backend self-study journey.

It is organized in three phases so the learning path stays practical and easy to navigate:

- Beginner: core Java syntax, OOP, collections, exceptions, file I/O, and a CLI capstone
- Intermediate / Junior: Maven, testing, debugging, SQL, JDBC or JPA, Spring Boot REST, and a capstone
- Pro / Advanced: architecture hardening, security, transactions, observability, Docker, CI/CD, and a stronger capstone

Each phase is split into topic folders. Every topic should contain code, short notes, and practice outputs. Weekly notes under `notes/` track what was learned, what still needs review, and what should be built next.

The repository is meant to keep learning evidence and working code together:

- code examples
- mini exercises
- weekly summaries
- screenshots or demos
- capstone projects

Recommended working style:

1. Study one topic at a time.
2. Type the code yourself instead of only reading examples.
3. Keep the output for every session small and concrete.
4. Write a short note after each session.
5. Commit progress frequently with meaningful messages.
