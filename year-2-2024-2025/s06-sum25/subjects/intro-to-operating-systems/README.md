# Intro To Operating Systems

This subject focuses on core operating-system concepts through hands-on C and C++ labs.

## Included Here

- `labs/process-synchronization`
  - thread cancellation, mutex, and semaphore exercises
- `labs/cpu-scheduling`
  - FIFO, Round Robin, and Shortest Job First implementations
- `labs/memory-management`
  - small memory-focused exercises from later labs

## Notes

Representative labs were selected from the semester workspace. Lecture slides and broader Linux setup exercises stay in the original local archive.
