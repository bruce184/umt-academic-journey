# About This Repository

This repository is designed to document my growth across university semesters in a clean and readable way.

The guiding mindset is:

> Meaning should be layered.

That means each folder level carries only the information it needs:

- the year folder carries the academic context
- the semester folder carries the time slice
- the subject folder carries the course context
- the project or lab folder carries the specific deliverable

This keeps the repository complete without becoming repetitive or messy.
