# Glossary

- `Widget tree`: the nested structure of widgets that defines the UI.
- `View model`: a layer that prepares state and actions for the UI.
- `Repository`: a layer that coordinates data access from local or remote sources.
- `Service`: a low-level data or platform dependency used by higher layers.
- `Persistence`: storing data so it survives app restarts.
- `Profile mode`: a Flutter build mode used to inspect runtime performance more realistically than debug mode.
