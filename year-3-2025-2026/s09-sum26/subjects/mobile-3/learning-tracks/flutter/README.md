# Flutter Learning Journey

A structured self-study repository that grows from Dart and Flutter fundamentals to junior-level app architecture and advanced delivery skills.

This repo now follows a phase-based learning journey instead of a single `projects/` bucket. The layout mirrors the same progression style used in the Java learning repo: each top-level phase has its own modules, notes, and future capstone space.

## Goals

- Build strong Dart and Flutter fundamentals before jumping into advanced architecture.
- Separate beginner, junior, and pro work so the repo stays easy to browse on GitHub.
- Keep practice code, weekly notes, roadmap files, and evidence in one place.
- Turn each phase into a clear study milestone with its own outputs and capstone.

## Learning Roadmap

### Phase 1: Beginner

- Duration: 6 weeks, 30 sessions
- Focus: setup, Dart basics, Flutter fundamentals, layout, forms, navigation, networking, state basics, first capstone
- Output: small Dart exercises, starter Flutter apps, and a beginner-level capstone
- Folder: `01_beginner/`

### Phase 2: Intermediate / Junior

- Duration: 6 weeks, 30 sessions
- Focus: Dart code quality, architecture, state flow, persistence, adaptive UI, testing, reliability, release mindset
- Output: a junior-level Flutter app with clearer architecture and stronger quality practices
- Folder: `02_intermediate_junior/`

### Phase 3: Pro / Advanced

- Duration: 6 weeks, 30 sessions
- Focus: scalable architecture, performance profiling, advanced data flow, platform integration, delivery, release engineering
- Output: an advanced app or module with stronger technical depth and portfolio/demo value
- Folder: `03_pro_advanced/`

## Repository Structure

```text

|-- 01_beginner/
|-- 02_intermediate_junior/
|-- 03_pro_advanced/
|-- notes/
|-- docs/
|-- templates/
|-- resources/
`-- assets/
```

### Top-Level Folders

- `01_beginner/`: beginner modules, Dart exercises, Flutter fundamentals, and beginner capstone work
- `02_intermediate_junior/`: architecture, persistence, testing, adaptive UI, and junior capstone work
- `03_pro_advanced/`: performance, platform integration, delivery, and advanced capstone work
- `notes/`: learning notes grouped by phase, then by week
- `docs/`: roadmap summaries, progress tracking, repo structure notes, and setup support files
- `templates/`: starter ideas for future Dart CLI or Flutter app practice
- `resources/`: curated reference links and study support materials
- `assets/`: screenshots, demos, and visual evidence grouped by phase
