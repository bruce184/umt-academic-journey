# Member 4 Demo Script

## Games
1. Open the hub and show that all seven games can be launched.
2. Start `Tic-Tac-Toe`, make a move, then save and exit.
3. Re-enter the same game and load the saved state.
4. Open `Snake` and show live movement plus score updates.
5. Open `Match-3`, make a valid swap, and show the score increase.
6. Open `Memory` and flip matching cards.
7. Open `Free Draw`, switch into palette mode, pick a color, and paint pixels.

## Admin
1. Log in as the seeded admin user.
2. Open the dashboard and point out total users, active users, sessions, and hottest game.
3. Open the users page and toggle one account enabled/disabled.
4. Promote or demote one account role.
5. Open the games page and update one board setting or timer.
6. Open the stats page and show popularity plus review distribution.
7. Open the API docs page and load the protected docs with the admin API key.

## Final checks
1. Confirm the app is running over HTTPS.
2. Confirm the docs page is blocked if the API key is removed.
3. Confirm saves still work after the admin changes.
