# Setup Checklist

## One-Time Setup

- Install JDK 21 LTS or JDK 25 LTS
- Install VS Code
- Install the Extension Pack for Java
- Install Git
- Confirm `java -version` works
- Confirm `javac -version` works

## Folder Setup

- Open the repository root in VS Code
- Work only inside the current phase folder
- Keep the terminal open in the correct folder for the current session

## Before Each Study Session

- Open only the module for the current session
- Close unrelated tabs and apps
- Open the required docs before coding
- Write down:
  - what you will study
  - what output you must finish
  - where you will stop if blocked
- Set a focused study block of about 90 to 120 minutes

## After Each Study Session

- Make sure the code runs
- Write a short note using the session template
- Save screenshots if the output is worth documenting
- Run `git status`
- Commit with a small, specific message
- Push when the session output is stable
