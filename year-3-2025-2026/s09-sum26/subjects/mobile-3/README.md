# Mobile 3

This subject folder tracks Mobile 3 coursework for Semester 9, curated from the local `3.JUNIOR/SEM9/MOBILE 3` folder.

## Current Local Materials

### Week 1 - Concurrent Programming

The first local week focuses on concurrent programming and platform-specific reading:

- Android multithreading material
- Kotlin coroutine material
- Android reading/report material
- iOS concurrent programming module material
- iOS reading/report material
- Android and iOS image browser examples

## Supporting Local Work

The folder also contains a separate Flutter learning journey repository. It is organized as a phase-based self-study track:

- `01_beginner/`: Dart basics, Flutter fundamentals, layout, forms, navigation, networking, and beginner capstone work
- `02_intermediate_junior/`: architecture, state flow, persistence, adaptive UI, testing, reliability, and junior capstone work
- `03_pro_advanced/`: scalable architecture, performance, platform integration, delivery, release engineering, and advanced capstone work
- `notes/`, `docs/`, `templates/`, `resources/`, and `assets/` for study support

## Curation Notes

Keep learning notes, selected source examples, app screenshots, and concise writeups. Do not commit Android Studio installation folders, emulator/runtime data, Gradle caches, generated build folders, or local SDK/tooling files.

The Flutter repository is currently better treated as a supporting standalone learning track unless it is intentionally absorbed later.
