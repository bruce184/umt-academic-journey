# Intermediate / Junior Roadmap Summary

## Goal

Move from "can build a small app" to "can build a cleaner, more structured, more reliable app".

## Weekly Focus

- Week 1: Dart deeper + code quality
- Week 2: architecture + state flow
- Week 3: persistence + caching
- Week 4: advanced UI + adaptive + navigation
- Week 5: testing + reliability
- Week 6: junior capstone + release mindset

## Main Output

- a junior-level app with clearer architecture
- local persistence or stronger API flow
- baseline testing and profile/release awareness
