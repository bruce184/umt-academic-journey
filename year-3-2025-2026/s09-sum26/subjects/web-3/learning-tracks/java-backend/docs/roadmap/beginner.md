# Beginner Roadmap

This phase focuses on building strong core Java fundamentals before moving into Maven, testing, databases, or frameworks.

## Phase Summary

- Duration: 5 weeks
- Sessions: 25
- Main goal: become comfortable writing small Java CLI programs from scratch
- Final output: one beginner-level capstone with clean structure and short documentation

## Week 1: Setup and Java Basics

- Goal: install the environment, run Java in the terminal and VS Code, and understand basic syntax
- Outputs:
  - first Java program
  - syntax practice files
  - mini calculator CLI

Topics:
- environment setup
- compile vs run
- variables and data types
- console input and output
- conditionals and loops
- methods and basic decomposition

## Week 2: OOP Foundations

- Goal: understand class and object thinking well enough to stop treating Java as only isolated statements
- Outputs:
  - class-based practice code
  - small OOP mini project such as `StudentManager` or `ProductManager`

Topics:
- classes and objects
- constructors
- `this`
- encapsulation
- access modifiers
- packages
- inheritance and overriding

## Week 3: Collections, Strings, and Exceptions

- Goal: manage real in-memory data safely and keep programs from crashing on simple input mistakes
- Outputs:
  - list or map practice files
  - one small app with add, remove, search, and input validation

Topics:
- arrays and `ArrayList`
- `HashMap` and `Set`
- `String`, `StringBuilder`, and parsing
- `try-catch`
- basic input validation

## Week 4: File I/O, Sorting, and Introductory Generics

- Goal: move from temporary in-memory exercises to small programs that can persist data in files
- Outputs:
  - file read/write examples
  - sorting examples
  - one console app that saves data to a file

Topics:
- text file read and write
- `Path` and `Files`
- `Comparable` and `Comparator`
- typed collections and generics basics

## Week 5: Beginner Capstone and Light Modern Java

- Goal: finish one demo-ready beginner CLI project and get a gentle introduction to lambda and stream syntax
- Outputs:
  - beginner capstone
  - project README
  - final 5-week summary note

Topics:
- basic lambdas
- basic streams
- beginner project structure
- capstone cleanup
- bug fixing
- demo preparation

## Recommended Capstone Options

- Task Manager CLI
- Expense Tracker CLI
- Library CLI
- Student Record Manager

If starting from zero, `Task Manager CLI` or `Expense Tracker CLI` is the safest choice.

## Session Workflow

For each session:

1. Read only the required docs for that topic.
2. Rebuild the example by typing it yourself.
3. Adjust the code a little to confirm understanding.
4. Write a short learning note.
5. Commit the session output.

## Not Included in Beginner

These topics are intentionally delayed until the next phase:

- Maven
- JUnit
- JDBC or JPA
- Spring Boot
- clean architecture
- database-backed applications
